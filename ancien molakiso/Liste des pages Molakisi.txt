Descriptions de chaque page
----------------------------

1. Page d'accueil : Pas besoin d'explications

2. Login : Page de connexion de chaque utilisateur

3. Inscription : Page d'inscription

4. Devenir tuteur :  Cette page donne des explications sur les avantages à devenir tuteur chez Molakisi (cfr Tuteria)

5. Demande d'ajout : Cette page comprend le formulaire qui permet aux intéressés de fournir les informations nécessaires à la candidature

6. Trouver un répétiteur : Dans cette page on donne un bref aperçu de comment rechercher un tuteur

7. Liste Tuteurs : Cette page liste tous les tuteurs de la plateforme

8. Tuteur :  Cette page montre en détail les informations concernant un tuteur en particulier

9. Demande de cours :  Une fois le client satisfait du profil du tuteur, cette page lui demande de remplir les informations liées à l'apprenant concerné.

create table t_candidature(
    id_cand int AUTO_INCREMENT not null,
    nom varchar(25),
    prenom varchar(25),
    postnom varchar(25),
    photo varchar(100),
    cour varchar(25),
    horaire varchar(25),
    cv varchar(100),
    CONSTRAINT pk_cand PRIMARY KEY (id_cand));
create table t_eleve(
    id_eleve int AUTO_INCREMENT not null,
    nom varchar(25),
    prenom varchar(25),
    postnom varchar(25),
    cour varchar(25),
    objetif varchar(25),
    horaire varchar(25),
    adresse varchar(25),
    classe varchar(100),
    CONSTRAINT pk_eleve PRIMARY KEY (id_eleve));
create table t_users(
id_users int AUTO_INCREMENT not null,
    name varchar(25),
    email varchar(25),
    pass varchar(25),
    CONSTRAINT pk_users PRIMARY KEY (id_users));
create table t_professor(
id_pro int AUTO_INCREMENT not null,
    name varchar(25),
    email varchar(25),
    pass varchar(25),
    CONSTRAINT pk_professor PRIMARY KEY (id_pro));
